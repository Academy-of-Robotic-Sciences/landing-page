Learning Tracks Overview

Robot Campus offers five specialized learning paths, each focusing on a critical aspect of humanoid robotics. Whether you're interested in mechanical design, electronics, software, AI, or systems integration, we have a track tailored for you.

Focus: CAD, 3D printing, mechanical systems

Master the art of designing robotic systems from concept to physical reality. Learn to create efficient, manufacturable designs that balance form and function.

Core Skills:

  • CAD modeling (Fusion 360, SolidWorks)
  • 3D printing and rapid prototyping
  • Material selection and stress analysis
  • Mechanism design and kinematics
  • Design for manufacturing (DFM)

Projects:

  • Design custom SO-101 joints
  • Create adaptive grippers
  • Develop modular robot chassis

âš¡ Robotic Hardware Engineerâ€‹

Focus: Electronics, sensors, actuators, embedded systems

Build the electronic nervous system of robots. From power management to sensor integration, become an expert in robotic hardware.

Core Skills:

  • Circuit design and PCB layout
  • Microcontroller programming
  • Sensor integration (IMU, force, vision)
  • Motor control and power systems
  • Communication protocols (I2C, SPI, CAN)

Projects:

  • Assemble and calibrate SO-101 arm
  • Build custom servo controllers
  • Design sensor fusion systems

ðŸ’» Robotic Software Engineerâ€‹

Focus: Control algorithms, ROS2, motion planning

Program robots to move with precision and intelligence. Master the software stack that brings robots to life.

Core Skills:

  • ROS2 framework and architecture
  • Kinematics and dynamics
  • Motion planning algorithms
  • Real-time control systems
  • Simulation (Gazebo, MuJoCo)

Projects:

  • Implement SO-101 inverse kinematics
  • Create trajectory planning systems
  • Develop URDF models and controllers

Focus: Machine learning, computer vision, behavioral AI

Train robots to learn and adapt using cutting-edge AI techniques. Bridge the gap between artificial intelligence and physical robotics.

Core Skills:

  • Deep learning for robotics
  • Computer vision and perception
  • Imitation and reinforcement learning
  • Sim-to-real transfer
  • LeRobot framework mastery

Projects:

  • Train pick-and-place with LeRobot
  • Implement visual servoing
  • Create adaptive grasping policies

Focus: Integration, architecture, deployment

Bring it all together. Design and deploy complete robotic systems that work reliably in the real world.

Core Skills:

  • System architecture design
  • Hardware-software integration
  • Safety and reliability engineering
  • Testing and validation
  • DevOps for robotics

Projects:

  • Full SO-101 task automation
  • Multi-robot coordination system
  • Production deployment pipeline

Can I Switch Tracks?â€‹

Absolutely! The tracks are designed to be complementary. Many successful roboticists eventually gain expertise across multiple tracks. Start with what excites you most, then expand your skills.
