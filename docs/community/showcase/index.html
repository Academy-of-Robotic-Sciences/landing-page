Submission Guidelines‚Äã

How to Submit Your Project‚Äã

  1. Document Your Build

    • Clear photos/videos
    • Technical specifications
    • Bill of materials
    • Assembly instructions

  2. Create Demo Video (3-5 minutes)

    • Project overview
    • Technical explanation
    • Live demonstration
    • Lessons learned

  3. Write Project Report

    • Problem statement
    • Design process
    • Challenges faced
    • Future improvements

  4. Submit via Form

    • Project Submission Form
    • Include all links
    • Select categories
    • Add team members

Evaluation Criteria‚Äã

Projects are evaluated on:

  • Innovation (25%): Novel approach or application
  • Technical Merit (25%): Engineering quality
  • Documentation (25%): Clarity and completeness
  • Impact (15%): Potential applications
  • Presentation (10%): Video and report quality
