# Projects Overview

Robot Campus projects are designed to provide hands-on experience with real robotic systems. Each project builds on the skills from our learning tracks while teaching practical implementation.

### ü¶æ [SO-101 Arm Project](/docs/projects/so-101-arm)
Build and program a 6-DOF robotic arm from the ground up. This flagship project covers mechanical assembly, electronics integration, software control, and AI-based manipulation.

**Skills Developed:**
- Servo motor control
- Inverse kinematics
- ROS2 integration
- Pick-and-place tasks

**Duration:** 2-3 months

- Model training
- Sim-to-real transfer

## Project Categories

### Beginner Projects
Perfect for getting started:
- **LED Cube**: Program an LED matrix display
- **Line Follower**: Build a simple mobile robot
- **Servo Tester**: Create a servo calibration tool
- **Sensor Dashboard**: Visualize sensor data

### Intermediate Projects
Build your skills:
- **2-DOF Arm**: Simple pick-and-place system
- **Mobile Base**: Differential drive platform
- **Vision System**: Object detection and tracking
- **Teleoperation**: Remote robot control

### Advanced Projects
Challenge yourself:
- **Quadruped Robot**: Walking gait development
- **Drone Control**: Autonomous flight system
- **Dual-Arm Coordination**: Bimanual manipulation
- **Swarm Robotics**: Multi-robot coordination
